In order to run the program: IndexBST.cpp

NOTE: Program is using C++ 11

1) First if needed to specify if code is using C++ 11 use:
	g++ -std=c++11 IndexBST.cpp -o IndexBST.o

   Else: 
	g++ IndexBST.cpp -o IndexBST.o

2) Make sure to have Text file with words you want to Index
	{EX: Test.txt}

3) Run Program as follows: 
	 ./IndexBST.o <file name with .txt extension here>
		
		{EX: ./IndexBST.o Test.txt}
